비교 연산자

    - 비교 연산자는
      좌변, 우변을 비교하여
      비교 결과가 맞을 경우 true,
      아닐 경우 flase를 반환한다.
      (boolean 자료형 == true, false)

은/는 의 배수가 맞는가?

복합 대입 연산자( +=, -=, *=, /= %=)

    - 대입 연산자(=) : 오른쪽 값을 왼쪽 변수에 대입
      기존 변수에 저장되어 있는 값에
      새롭게 대입되는 값의 산술 연산 결과를 대입

논리 연산자(&&, ||, !)

    &&(AND) 연산자 -> A && B

    - A와 B 모두 true인 경우에만 
      A && B 연산 결과가 true

    ex) 104는 100 이상의 숫자 이면서 짝수인가?
    -> (104>=100) && (104%2===0) -> true

    -A 와 B 모두 false인 경우에만
    A || B 연산 결과가 false

    ! (NOT) 연산자
    - 논리 값을 반대로 바꾸는 연산자
      (논리 부정 연산자)
